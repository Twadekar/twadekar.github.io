Project: Car Price Prediction (Machine Learning)

View Project

Problem

Provide reliable used Toyota car price estimates for marketplace listings.

Dataset

Used car dataset containing features such as brand, year, mileage, fuel type, transmission, and selling price.

Tools

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

Key Steps

  • Feature engineering (age, log-mileage, encoded categorical variables)
  • Model training using regression models
  • Model evaluation using RMSE and R²

Impact

Achieved R² score of 0.82 indicating strong predictive performance.
